# vue-element-v1

#### 安装
```
npm install
```

#### 运行
```
npm run serve
```

#### 打包
```
npm run build
```
**目涉及的技术主要有：** 

ES6 、vue 、vuex 、vue-router 、vue-cli 、axios 、stylus 、element-ui 、mockjs 

项目是个人学习Vuejs看了相关资料一个月多后完成的，有待完善。大都用原始的element-ui组件、页面单一、 没有过多复杂的配置项和过多的封装【相信新手看过其他成型的成品会有感受】
https://github.com/fucaiwei/vue-element-v1
